# Openforti pass + opt

## Install 
* 
        https://github.com/adrienverge/openfortivpn
*
        ~ pip install - r requirements.txt
## Create
    ~ touch config.json

```json
{
    "host": "",
    "port": "",
    "user": "",
    "password": "",
    "opt": "otpauth://",
    "trusted-cert": ""
}
```
## Run
    ~ python main.py
